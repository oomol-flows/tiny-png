# Tiny PNG

English | [Chinese](./README_zh-CN.md)

## Introduction

> [HomePage](https://tinify.com/)

TinyPNG is an advanced image compression technology that focuses on reducing the size of PNG and JPEG files without compromising quality. It uses smart lossy compression techniques to reduce irrelevant information in images while maintaining visual quality.

- **Lossless Compression**: TinyPNG uses the latest technology to significantly reduce file size while ensuring image quality.
- **Batch Processing**: You can upload and compress multiple image files at once.
- **API Support**: Provides a powerful API interface for developers to easily integrate compression functionality into their applications or workflows.
- **Cross-platform Tools**: Supports multiple operating systems and offers various usage methods including command-line tools and plugins.

## Features

- [Dashboard](https://tinify.com/dashboard/api)
- Supports PNG and JPEG formats
- Optional metadata retention
- Offers both free and paid usage modes
- Free monthly quota of 500 images (account registration required)
