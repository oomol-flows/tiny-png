# Tiny PNG

[English](./README.md) | Chinese

## Introduction

> [HomePage](https://tinify.com/)

TinyPNG 是一种先进的图片压缩技术，专注于在不损失画质的情况下减小PNG和JPEG文件的大小。它利用智能有损压缩技术减少图像中的无关信息，同时保持视觉质量不变。

- **无损压缩**：TinyPNG 使用最新技术，在保证图像质量的同时显著减小文件大小。
- **批量处理**：可以一次性上传并压缩多个图片文件。
- **API支持**：提供强大的API接口，方便开发者将压缩功能集成到自己的应用或工作流中。
- **跨平台工具**：支持多种操作系统，并且提供了命令行工具、插件等多种形式的使用方式。
